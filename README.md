## What It Is

Game News Round-Up is a curated daily digest of what is happening in the games industry. Every story links straight back to the outlet that broke it, so you can read the full thing wherever it came from.

The game news is hand-curated, without the assistance of AI. The stories are found, chosen, and written by a human being.

This is a free service, intentionally designed to not require any sign-up, cookies, tracking and contains zero advertisements.

## What You'll Find

Each day is sorted into a few clear sections so you can jump to whatever you care about:

- **Game News**
  - Announcements, releases, platform updates, and the big headlines.
- **Free Games / Subscription**
  - What's new and leaving on PS Plus, Game Pass, Epic, Prime Gaming, and the rest.
- **Business**
  - Sales figures, deals, studio news, and the money side of the industry.
- **Rumors**
  - Stories that are still up in the air, kept separate and clearly labelled with how confident we are in them.

You can browse the latest round-up on the home page, dig through the full archive by date, and tweak the look with a different accent colour or a more compact layout if you prefer.

## Feeds and Data

If you would rather have the round-up come to you, or you want to build something with it, every day is also published in open formats:

- An **RSS feed** you can drop into any reader.
- A **JSON API** for bots, tools, and tinkering.
- A **Copy for Slack** button on every round-up, so you can paste the day straight into a channel on Slack.

See the [Feeds page](https://gamenewsroundup.co.uk/feeds/) for the links. 

## About This Repository

This repository is the website itself. It is a deliberately simple, fast, static site: plain HTML, CSS, and a small amount of JavaScript, served by GitHub Pages with no server or database behind it. Each day's round-up is stored as a structured file and the site is generated from those files, which keeps the whole thing lightweight and quick to load.

If you are just here to read the news, head to [gamenewsroundup.co.uk](https://gamenewsroundup.co.uk). If you are curious about how it is put together, feel free to look around the code.
