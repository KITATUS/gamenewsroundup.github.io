Nvidia tells users to urgently update their GPU drivers to fix eight ‘highly severe’ vulnerabilities

  • The security risks affect Nvidia’s GeForce, RTX, Quadro, NVS and Tesla products, and if not patched could lead to unauthorised attacks on users’ PCs.
  • There are two main types of vulnerabilities exposed by the risks identified. The first “could allow a privileged attacker to escalate permissions”, while the other is “a vulnerability in the user mode layer, where an unprivileged regular user can cause an out-of-bounds read”.
  • Nvidia says this can result in “remote code execution, denial of services, escalation of privileges, information disclosure and data tampering” if not addressed.
